# Michael's Corner

Personal brand HQ. Where video scripts, thumbnails, publishing checklists, and production notes live.

## Folders

- `scripts/` — video scripts and outlines (one file per video)
- `ideas/` — raw idea backlog, hooks library, title formulas
- `thumbnails/` — thumbnail concepts and exported files
- `checklists/` — pre-publish checklists per platform + language
- `production/` — camera setup, lighting, scenery, audio notes
- `brand/` — colors, fonts, voice, visual identity
- `docs/` — public-facing personal site (GitHub Pages serves from here)

## Platforms

YouTube (long-form), YouTube Shorts, TikTok, Instagram Reels, Instagram posts/stories, LinkedIn.

## Languages

Czech (native) and English (B2). See `checklists/` for per-platform language strategy.

## Public site

Lives in `site/index.html`. Deployed via GitHub Pages at `https://off-plate.github.io/michaels-corner/`.
Link-only — no SEO push, no auth. Same model as the EduJoy audit.
