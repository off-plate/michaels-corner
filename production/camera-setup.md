# Camera Setup

Talking-head defaults. For full scene reference (lighting + composition + diagram), see `set-design.md`.

## Talking head (default)
- **Focal length**: 50mm – 85mm
- **Aperture**: f/2.0 – f/2.8 (shallow depth, background blur)
- **Focus**: AF single point, locked on eyes
- **Resolution**: 4K
- **Frame rate**: 24fps or 25fps (PAL region — matches CZ standard)
- **White balance**: 5000K – 5600K (manual, set per location — no AWB)
- **ISO**: 100 – 400 (keep low, raise only if light forces it)
- **Shutter**: 1/50 (matches 25fps double-rule)
- **Picture profile**: flat / log if grading, standard if not

## Framing
- Medium close-up, chest-up
- Subject angled 30°–45° away from camera
- Looking at off-camera interviewer, not into the lens
- Rule of thirds — leave negative space for overlays

## Phone (Shorts / on-the-go)
- 4K 30p, 1x lens (not ultrawide — face distortion)
- Native camera app, not in-app capture (TikTok, IG)
- Lock exposure + focus before recording

## Audio
- **Levels**: peaks at -12 to -6 dB. Never let it hit 0.
- **Backup**: always record on-camera audio as safety track
- 5 seconds of room tone before each session

## Storage / backup
- SD card → laptop → cloud, every session
- Don't delete cards until cloud copy confirmed
- One folder per video: `YYYY-MM-DD_slug/`

## Pre-roll checklist
- [ ] Battery > 50%
- [ ] Card formatted, has space
- [ ] Mic on, levels checked
- [ ] Focus locked on eyes
- [ ] Red dot confirmed before talking
