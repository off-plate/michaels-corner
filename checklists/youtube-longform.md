# YouTube Long-form — Pre-publish Checklist

## Pre-production
- [ ] Idea validated (hook works, audience clear)
- [ ] Script written and read out loud once
- [ ] Thumbnail concept sketched before filming
- [ ] B-roll list ready

## Filming
- [ ] Camera settings checked (see `production/camera-setup.md`)
- [ ] Audio levels checked (peaks around -12 to -6 dB)
- [ ] Lighting locked
- [ ] Background tidy (see `production/scenery.md`)
- [ ] Backup take of intro and outro

## Editing
- [ ] Hook in first 5 seconds — re-watch with sound off, does it grip?
- [ ] Pacing: cut every dead beat
- [ ] Captions burned in for first 30s minimum
- [ ] Chapters added (every 60–90s)
- [ ] End screen with 1 video + subscribe

## Metadata
- [ ] Title — under 60 chars, curiosity gap, no clickbait lies
- [ ] Description — first 2 lines are the hook (these show in search)
- [ ] Tags — 5–8 relevant
- [ ] Thumbnail — readable at 120px width, max 3 visual elements
- [ ] Pinned comment ready

## Language strategy
- **CZ video**: Czech title, CZ description, CZ thumbnail text. Add CC for accessibility.
- **EN video**: English title, EN description, EN thumbnail text. Auto-translate CC to CZ if relevant audience.
- Decide language by topic, not by mood. Don't mix in same video.

## Publish
- [ ] Schedule (don't publish blindly — pick a time)
- [ ] Cross-post short clip to Shorts / Reels / TikTok within 48h
- [ ] LinkedIn post if topic fits
- [ ] Add to relevant playlist
