# LinkedIn — Pre-publish Checklist

Mostly English. Mostly text or text + image. Sometimes video.

## Text post
- [ ] Hook in line 1 (this is all most people see before "see more")
- [ ] Line 2 is a single-line follow-up that earns the click
- [ ] Body: short paragraphs, one idea per paragraph
- [ ] No corporate-speak, no "I'm humbled to announce"
- [ ] Specific > generic (numbers, names of things, real outcomes)
- [ ] One CTA at the end (question to drive comments works best)

## Image / carousel post
- [ ] Image at 1200×1200 or PDF carousel (4:5 portrait performs best)
- [ ] Carousel: 6–10 slides, each with one takeaway
- [ ] First slide = hook, last slide = CTA
- [ ] Brand consistent (see `brand/visual-identity.md`)

## Video post
- [ ] Subtitles burned in (LinkedIn autoplay is muted)
- [ ] First 3 seconds must work without sound
- [ ] Native upload (don't link out to YouTube — LinkedIn punishes external links)
- [ ] Keep under 90 seconds unless content really needs more

## Language
- LinkedIn = mostly English by default (broader reach, professional audience)
- Czech LinkedIn post if the topic is local market specific
- Don't bilingual-post unless audience is genuinely split

## Engagement protocol
- [ ] Reply to every comment in first 60 minutes (algorithm window)
- [ ] No edits in first 30 min (edits suppress reach)
- [ ] No external links in post body — put them in first comment

## Publish
- [ ] Post Tue–Thu, 8–10am CET for B2B
- [ ] Don't post weekends unless intentional
