# Shorts / TikTok / Reels — Pre-publish Checklist

Vertical short-form (9:16). Same content, three platforms, slight tweaks each.

## Hook (first 1–2 seconds)
- [ ] Visual movement in frame 1
- [ ] First spoken word is intriguing or punchy
- [ ] Text overlay reinforces the hook
- [ ] No slow intro — drop into the value immediately

## Pacing
- [ ] Cut every 1–3 seconds
- [ ] No dead air
- [ ] One idea per video. Don't stuff.
- [ ] Length: 15–45s sweet spot. Up to 60s only if needed.

## Captions
- [ ] Burned-in captions, large, high contrast
- [ ] Native font of each platform avoided (looks low-effort)
- [ ] Keywords in caption text for searchability

## Audio
- [ ] Voice clear, no background noise
- [ ] Trending sound used only if it doesn't bury your voice
- [ ] Check each platform's trending sounds the day of posting

## Per-platform tweaks

### YouTube Shorts
- [ ] Title: keep clear, no hashtag spam
- [ ] Pin a comment that drives to long-form
- [ ] Add `#Shorts` in description (optional, low impact)

### TikTok
- [ ] 3–5 hashtags max, mix broad + niche
- [ ] Reply to top comments within 1h of posting
- [ ] Cover image — TikTok auto-picks bad ones, set one manually

### Instagram Reels
- [ ] Cover image set manually (matters for grid)
- [ ] Caption: hook in line 1, then context
- [ ] Use 3–5 hashtags inside caption, not in comment
- [ ] Share to your story after posting

## Language strategy
- **CZ short**: Czech captions + caption text. Czech audience finds it.
- **EN short**: English captions + caption text. International reach.
- One language per video — never bilingual captions, it kills both audiences.
- Same idea CAN be reshot in the other language (re-record, don't dub).

## Publish
- [ ] Post to all 3 platforms within 24h
- [ ] Don't post the same TikTok-watermarked video to Reels (kills reach)
- [ ] Export raw 9:16 without watermark for cross-posting
