# Thumbnails

One folder per video: `YYYY-MM-DD_slug/`
Inside: concepts (sketches, references), working files, final export.

## File naming
- `concept-1.png`, `concept-2.png` — early ideas
- `wip.psd` / `wip.fig` — working file
- `final.png` — 1280×720, < 2MB (YouTube spec)
- `final-1x1.png` — for IG grid if cross-posting

## Rules
See `brand/visual-identity.md` for thumbnail style rules. Don't skip.
