# Instagram (Posts + Stories) — Pre-publish Checklist

For static posts, carousels, and stories. Reels live in `shorts-tiktok-reels.md`.

## Static post / carousel
- [ ] First slide is the hook (text + visual)
- [ ] Carousel: each slide stands alone but pulls to the next
- [ ] Last slide has a CTA (save, share, follow)
- [ ] Caption: hook in line 1, body in 2–4 short paragraphs
- [ ] 3–5 hashtags inside caption
- [ ] Alt text added (accessibility + minor SEO)

## Stories
- [ ] Native stickers (poll, question, quiz) — they boost reach
- [ ] No more than 5 frames per story sequence
- [ ] Add link sticker if driving traffic
- [ ] Highlight cover prepared if it's going to a highlight

## Grid aesthetic
- [ ] New post fits visually with last 8 posts (preview in planner)
- [ ] Don't post 3 of the same type back to back

## Language strategy
- **CZ post**: Czech caption, Czech hashtags (#cesko, #praha, niche-specific)
- **EN post**: English caption, English hashtags
- Bilingual caption (CZ first, --- divider, EN below) is OK for posts that genuinely serve both audiences. Decide deliberately.

## Publish
- [ ] Post at peak time for your audience (check Insights)
- [ ] Share to story immediately after posting
- [ ] Reply to every comment in the first hour
